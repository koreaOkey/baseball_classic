## ADDED Requirements

### Requirement: 소셜 로그인
시스템은 카카오 계정과 Apple 계정으로 로그인할 수 있어야 한다(SHALL). 그 외 로그인 수단은 본 차수에서 지원하지 않는다.

#### Scenario: 카카오 로그인 성공
- **WHEN** 사용자가 로그인 화면에서 카카오 버튼을 누르고 카카오 인증을 완료한다
- **THEN** 시스템은 사용자 세션을 생성하고 커뮤니티 탭에 진입시킨다

#### Scenario: Apple 로그인 성공
- **WHEN** 사용자가 로그인 화면에서 Apple 버튼을 누르고 Apple 인증을 완료한다
- **THEN** 시스템은 사용자 세션을 생성하고 커뮤니티 탭에 진입시킨다

#### Scenario: 익명/게스트 로그인 금지
- **WHEN** 사용자가 소셜 로그인 없이 커뮤니티 기능에 접근하려 한다
- **THEN** 시스템은 로그인 화면으로 이동시키고 익명 사용은 허용하지 않는다

### Requirement: 커뮤니티 탭 로그인 게이트
시스템은 홈·라이브·ME 탭은 로그인 없이 허용하되(SHALL), 커뮤니티 탭 진입 시에만 로그인을 강제해야 한다(SHALL).

#### Scenario: 비로그인 사용자의 홈 탭 사용
- **WHEN** 로그인하지 않은 사용자가 홈 탭을 연다
- **THEN** 시스템은 로그인을 요구하지 않고 기존 홈 화면을 표시한다

#### Scenario: 비로그인 사용자의 커뮤니티 탭 진입
- **WHEN** 로그인하지 않은 사용자가 커뮤니티 탭을 누른다
- **THEN** 시스템은 로그인 화면을 표시하고 로그인 후 커뮤니티 홈으로 이동시킨다

### Requirement: 사용자 프로필
시스템은 로그인한 사용자에 대해 닉네임·응원팀·프로필 이미지 URL을 저장해야 한다(SHALL).

#### Scenario: 최초 로그인 시 응원팀 선택
- **WHEN** 사용자가 최초 로그인을 완료한다
- **THEN** 시스템은 응원팀 선택을 요청하고, 선택한 팀을 프로필에 저장한다

#### Scenario: 프로필 식별자 유지
- **WHEN** 사용자가 재로그인한다
- **THEN** 시스템은 동일한 내부 user_id로 기존 프로필을 복원한다

### Requirement: 가이드라인 동의
시스템은 최초 로그인 시 이용약관·개인정보처리방침·커뮤니티 가이드라인에 대한 동의를 받아야 한다(MUST).

#### Scenario: 최초 로그인 시 약관 동의
- **WHEN** 사용자가 최초 로그인을 시도한다
- **THEN** 시스템은 3종 약관 동의 화면을 표시하고 모두 동의하지 않으면 가입을 완료하지 않는다

#### Scenario: 약관 개정 시 재동의
- **WHEN** 약관이 개정되어 사용자가 로그인한다
- **THEN** 시스템은 재동의 화면을 표시하고 동의 전까지 커뮤니티 기능을 제한한다
