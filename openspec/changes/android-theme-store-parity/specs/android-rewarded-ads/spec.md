## ADDED Requirements

### Requirement: 리워드 광고로 테마 잠금 해제
사용자가 Ad-Reward 카테고리 테마의 "광고 보고 받기" 버튼을 탭하면 리워드 광고를 로드하여 표시하고, 시청 완료 시 해당 테마를 잠금 해제하여 자동 적용해야 한다.

#### Scenario: 광고 시청 완료 후 테마 잠금 해제
- **WHEN** 사용자가 잠긴 Ad-Reward 테마의 "광고 보고 받기"를 탭하고 광고를 끝까지 시청
- **THEN** 해당 테마가 잠금 해제되고 즉시 적용되며, 로컬 저장소에 해제 이력이 저장된다

#### Scenario: 광고 로드 실패
- **WHEN** 네트워크 오류 등으로 광고 로드가 실패
- **THEN** 테마는 잠긴 상태를 유지하고, 로딩 표시가 해제된다

#### Scenario: 광고 로딩 중 중복 탭 방지
- **WHEN** 광고가 로딩 중인 상태에서 사용자가 다시 "광고 보고 받기"를 탭
- **THEN** 중복 로드 없이 기존 로딩이 계속된다
