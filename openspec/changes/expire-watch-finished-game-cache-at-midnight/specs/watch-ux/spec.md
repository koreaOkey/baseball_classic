## ADDED Requirements

### Requirement: 종료 경기 캐시 자정 만료
워치 화면과 타일은 종료된 경기의 저장된 상태를 다음날 KST 자정 이후 만료 처리해야 한다(SHALL).

#### Scenario: 종료 당일에는 최종 경기 표시 유지
- **GIVEN** 워치에 종료된 경기 상태가 저장되어 있고
- **AND** 현재 시각이 해당 경기 상태를 받은 날의 KST 23:59 이전일 때
- **WHEN** 사용자가 워치 앱 또는 타일을 열면
- **THEN** 워치는 저장된 최종 경기 상태를 표시한다

#### Scenario: 다음날 자정 이후 경기 없음 표시
- **GIVEN** 워치에 종료된 경기 상태가 저장되어 있고
- **AND** 현재 시각이 해당 경기 상태를 받은 날의 다음날 KST 00:00 이후일 때
- **WHEN** 사용자가 워치 앱 또는 타일을 열면
- **THEN** 워치는 저장된 종료 경기 상태를 만료 처리한다
- **AND** 진행 중인 경기가 없는 UI를 표시한다

#### Scenario: 예정 경기 시작 전에는 이전 종료 경기 미표시
- **GIVEN** 이전 경기의 종료 상태가 만료되었고
- **AND** 오늘 내 팀 경기가 아직 라이브 상태가 아닐 때
- **WHEN** 사용자가 워치 앱 또는 타일을 열면
- **THEN** 워치는 이전 종료 경기 결과를 표시하지 않는다
- **AND** 진행 중인 경기가 없는 UI를 표시한다
