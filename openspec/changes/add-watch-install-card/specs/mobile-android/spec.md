## ADDED Requirements

### Requirement: 설정 화면 최상단 워치 앱 상태 카드

Android 폰 앱의 설정 화면은 화면 타이틀("설정") 바로 다음, 다른 어떤 설정 항목보다 위에 워치 동반 앱 상태 카드를 노출해야 한다(SHALL). 카드는 워치 동반 앱 상태에 따라 다음과 같이 표현되어야 한다:

- **PAIRED_NONE**: 회색 톤 카드 + "워치를 폰과 페어링해 주세요" 안내 + Wear OS 페어링 도움말 진입점. 설치 버튼은 비활성 또는 숨김.
- **PAIRED_NO_APP**: 워치 아이콘 위 빨간 경고 배지 + "워치 앱 설치 필요" 제목 + Play Store 안내 문구 + [워치 앱 설치하기] 기본 버튼 + [연결 확인] 보조 버튼.
- **INSTALLED**: 초록 체크 + "워치 앱 설치됨" 제목 + 컴팩트한 [다시 확인] 보조 버튼. 카드 높이는 다른 두 상태보다 작아야 한다.

카드의 기본 버튼 색상은 사용자가 선택한 응원 팀 컬러(`teamTheme.primary`)를 따라야 한다(SHALL). 빨간 경고 배지와 초록 성공 체크는 팀컬러와 무관한 고정 색을 사용한다.

#### Scenario: 워치를 페어링하지 않은 사용자가 설정 탭을 열었을 때

- **WHEN** 페어링된 워치가 없는 사용자가 설정 탭에 진입
- **THEN** "설정" 타이틀 바로 아래에 PAIRED_NONE 상태 카드가 가장 먼저 표시되고, 페어링 안내 문구가 노출된다

#### Scenario: 워치는 있지만 앱이 설치되지 않은 사용자가 설정 탭을 열었을 때

- **WHEN** 워치를 페어링했지만 워치에 야구봄 앱을 아직 설치하지 않은 사용자가 설정 탭에 진입
- **THEN** "설정" 타이틀 바로 아래에 PAIRED_NO_APP 상태 카드가 가장 먼저 표시되고, [워치 앱 설치하기] 버튼이 응원 팀 컬러로 노출된다

#### Scenario: 워치 앱이 정상 설치된 사용자가 설정 탭을 열었을 때

- **WHEN** 워치에 야구봄 앱이 설치된 사용자가 설정 탭에 진입
- **THEN** "설정" 타이틀 바로 아래에 INSTALLED 상태의 컴팩트 카드가 가장 먼저 표시되고, 초록 체크와 함께 "워치 앱 설치됨" 문구가 노출된다

#### Scenario: 응원 팀을 변경한 직후

- **WHEN** 사용자가 응원 팀을 변경
- **THEN** 카드의 기본 버튼 색상이 새 팀의 컬러로 즉시 갱신된다

### Requirement: Play Store 설치 유도 흐름

PAIRED_NO_APP 상태에서 [워치 앱 설치하기] 버튼은 폰의 Play Store 앱을 직접 호출하여 야구봄 앱 상세 페이지를 열어야 한다(SHALL). 카드에는 사용자가 설치 대상 기기 선택 드롭다운에서 워치를 직접 선택해야 한다는 안내 문구가 함께 노출되어야 한다.

#### Scenario: 사용자가 [워치 앱 설치하기] 버튼을 누름

- **WHEN** PAIRED_NO_APP 상태에서 사용자가 [워치 앱 설치하기] 버튼을 누름
- **THEN** Play Store 앱이 폰 앱 상세 페이지로 열리고, 카드 하단에는 "Play Store에서 설치 대상 기기를 워치로 선택해 주세요" 안내가 사용자에게 사전 노출되어 있다

#### Scenario: Play Store 앱이 설치되어 있지 않은 환경

- **WHEN** Play Store 앱이 설치되지 않은 단말에서 사용자가 [워치 앱 설치하기] 버튼을 누름
- **THEN** 시스템은 일반 브라우저로 Play Store 웹 페이지를 열거나, 설치 불가 상태를 사용자에게 알린다

### Requirement: 워치 앱 capability 선언

워치 앱은 폰 앱이 설치 여부를 감지할 수 있도록 고정된 capability 식별자를 선언해야 한다(SHALL). 이 capability 는 워치 앱 설치만으로 시스템에 등록되며, 워치 앱이 한 번도 실행되지 않은 상태에서도 폰 앱이 이를 발견할 수 있어야 한다(MUST).

#### Scenario: 워치 앱 설치 직후

- **WHEN** 사용자가 워치에 야구봄 앱을 설치한 직후, 워치 앱을 한 번도 실행하지 않음
- **THEN** 폰 앱은 capability 조회 결과로 워치 앱의 존재를 감지하고 INSTALLED 상태로 분류한다
