## ADDED Requirements

### Requirement: Wear OS 팀명 한글 마스코트명 표시
워치 경기 화면에서 팀명은 영문 약어가 아닌 한글 마스코트명(예: 랜더스, 트윈스)으로 표시되어야 한다(SHALL).

#### Scenario: 영문 팀 코드가 전달된 경우
- **WHEN** 경기 데이터에 팀명이 영문 약어(예: "SSG", "LG")로 전달되면
- **THEN** 화면에는 해당하는 한글 마스코트명(예: "랜더스", "트윈스")으로 표시된다

### Requirement: 경기 라이브 중 화면 꺼짐 방지
경기가 라이브 상태일 때 워치 화면이 시스템 타임아웃으로 꺼지지 않아야 한다(SHALL).

#### Scenario: 경기 라이브 중 화면 유지
- **WHEN** 경기 상태가 라이브(isLive=true)이면
- **THEN** 워치 화면이 켜진 상태를 유지한다

#### Scenario: 경기 종료 시 화면 유지 해제
- **WHEN** 경기가 종료(isLive=false)되면
- **THEN** 화면 유지가 해제되어 시스템 기본 타임아웃이 적용된다
